# Identity & Access Risk Analytics System

## Overview
Developed a cybersecurity-focused analytics solution to identify risky user access behavior, login anomalies, and privilege misuse using Python, SQL Server, and Power BI.

## Tools Used
- Python
- Pandas
- Faker
- SQL Server
- SSMS
- Power BI

## Key Features
- Generated and analyzed 50,000+ security events
- Built custom risk scoring engine
- Detected country and device anomalies
- Created user risk profiles
- Developed interactive Power BI dashboard

## Dashboard KPIs
- Total Events
- Critical Events
- Country Anomalies
- Device Anomalies

## Analytics Performed
- Risk distribution analysis
- Department risk assessment
- Country anomaly detection
- High-risk user identification
